# Compact Projects Section with Dialogs

This file contains the new compact projects section that uses cards with dialogs.

Key features:

- 3-column grid layout (responsive)
- Compact cards showing title, description, and 3 key tech badges
- "View Details" button on each card
- Click to open full project details in a modal dialog
- Background blur when dialog is open
- Scrollable dialog content

The compact cards show:

- Badge indicating project type
- Star icon
- Project title
- Short description (line-clamped)
- 3 main tech stack badges
- View Details button

The dialog shows full project details:

- Complete overview
- Problem solved
- Key achievements
- Full tech stack
- Embedded YouTube demo
- GitHub link button
